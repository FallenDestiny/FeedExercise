# FeedExercise

FeedExercise React Native test exercise
### Development
#### Dependencies
- react-native 0.61.5
- react 16.9.0
- redux 4.0.5
- react-navigation 4.1.1
#### Initial setup
1. Clone the repo
```
git clone git@github.com:FallenDestiny/FeedExercise.git
```
2. Install dependencies and perform initial app setup
```
npm install
```
2. Install cocoapods
```
cd ios
pod install
```
3. Run the app
```
npx react-native run-ios
```
